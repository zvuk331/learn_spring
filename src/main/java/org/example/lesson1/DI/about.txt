IoC - Inversional of Control   - Даёт разрешения на создания объектов, теперь создавать может не только программист
ну и Spring
DI - Dependency Injection   - Внедрение зависимостей, управляет зависимостями объекта. Spring сам создаёт и внедряет Бины.

@Component - обозначает класс для спринга как компонент
@Autowired - автоматическое внедрение зависимости
@PostConstruct - Отмечает метод, который должен выполниться до создания бина. Inin method
@PreDestroy - Метод, который выполнится после смерти бина. Destroy method.

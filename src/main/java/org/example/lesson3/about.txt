POJO (Plain Old Java Object) Class - это класс удовлетворяющий условиям:
приватные поля, геттеры и сеттеры, конструкттор без аргументов и т.д.

При CascadeType.ALL - вся связанные поля будут автоматически обновляться/удаляться/добавляться.

FetchType.EAGER - нетерпеливая загрузка, сразу загружаются все связанные данные.

FetchType.LAZY - ленивая загрузка, данные загружается при обращении к ним.


Если убрать CascadeType.ALL чтобы не удалялись все связанные данные, тогда нужно использовать вместо
метода save() - persist(). CascadeType.PERSISTANCE будет отвечать за сохранение всех связанных данных в БД.
